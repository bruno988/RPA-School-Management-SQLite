# RPA-School-Management-SQLite

Título: Automação de Gestão Escolar com RPA e SQLite.

Objetivo: Processar uma base de dados de 50 usuários (simulando 500) para triagem de perfis (Alunos, Professores, Administrativos).

Tecnologias: Power Automate Desktop, SQL (SQLite), ODBC Driver.

Como rodar: Explique que é necessário o DSN configurado e o driver instalado.
